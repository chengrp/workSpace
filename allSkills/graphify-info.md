# Graphify 安装信息

## GitHub 信息
- **仓库地址**: https://github.com/safishamsi/graphify
- **作者**: safishamsi
- **Star**: 给个 star 支持一下 https://github.com/safishamsi/graphify

## 安装信息
- **PyPI 包名**: `graphifyy`（注意是两个 y）
- **安装命令**: `pip install graphifyy && graphify install`
- **当前版本**: 0.3.11

## 更新方式
```bash
pip install --upgrade graphifyy && graphify install
```

## 核心功能
将代码、文档、论文、截图转换为可查询的知识图谱
- 19 种编程语言 AST 解析
- 多模态支持（PDF、Markdown、图片）
- 交互式 HTML 可视化
- Token 节省高达 71.5x
