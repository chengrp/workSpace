# 张小龙完整时间线

## 一、早年经历与教育背景

| 年份 | 事件 | 意义 |
|------|------|------|
| 1969年12月3日 | 出生于湖南省邵阳市洞口县高沙镇，祖籍湖南省邵阳市邵东市魏家桥镇 | "微信之父"的诞生地 |
| 1987年 | 考入华中科技大学（原华中工学院）电信系 | 开始专业学习生涯 |
| 1989年 | 开始学习编写程序 | 编程生涯的起点 |
| 1994年 | 获华中科技大学硕士学位 | 完成高等教育 |

## 二、Foxmail时期

| 年份 | 事件 | 意义 |
|------|------|------|
| 1994年 | 南下广州，从事软件开发工作 | 进入互联网行业 |
| 1997年1月 | FoxMail 1.0 beta（英文版）发布，在winsite上公布 | 国产优秀电子邮件客户端软件诞生 |
| 1997年 | 时任《电脑报》记者李学凌描述："只要你站在黄庄路口，大喊一声，我是Foxmail张小龙，一定会有一大群人围上来，让你签名" | 张小龙在IT圈知名度达到顶峰 |
| 1998年9月 | 时任金山总经理雷军联系张小龙，希望以15万元收购Foxmail，但因事务缠身未果 | 错失第一次被收购机会 |
| 2000年4月18日 | 以1200万元价格将Foxmail卖给博大公司，任公司副总裁 | 获得第一桶金，互联网泡沫时期的成功变现 |
| 2001年 | Foxmail用户量达到200万，超过当时腾讯QQ的10万用户 | 成为国内最受欢迎的邮件工具之一 |

## 三、腾讯时期

| 年份 | 事件 | 意义 |
|------|------|------|
| 2005年3月16日 | 腾讯收购Foxmail，张小龙加盟腾讯，担任广州研发部总经理 | 正式加入腾讯帝国 |
| 2005年4月 | 腾讯广州研究院成立，主要负责邮件相关业务 | QQ邮箱团队组建 |
| 2006年底 | 张小龙率领团队用精简、轻便思路设计新版QQ邮箱 | 开始重塑QQ邮箱 |
| 2007年 | 新版QQ邮箱上线 | QQ邮箱转型成功 |
| 2008年3月 | QQ邮箱成为国内使用人数最多的邮箱产品 | QQ邮箱成为腾讯口碑产品 |
| 2008年 | QQ邮箱获得腾讯"七星级产品奖"，张小龙团队获年度创新大奖 | 产品获得官方认可 |
| 2010年8月2日 | 被任命为腾讯副总裁 | 进入腾讯高层管理 |
| 2010年11月19日 | 向马化腾发邮件建议开发移动社交软件，微信项目正式启动 | 微信诞生的关键决策 |
| 2010年11月20日 | 微信正式立项，由张小龙负责 | 微信项目正式启动 |
