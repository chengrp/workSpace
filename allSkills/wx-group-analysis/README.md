# 微信群数据分析 Skill

基于 wechat-cli 的微信群数据分析工具，自动生成多维度群分析报告。

## 功能特性

- ✅ **5种报告类型**: 群日报、成员贡献榜、内容质量报告、成员成长报告、综合月报
- ✅ **智能时间解析**: 支持今天、昨天、本周、上周、本月、上月等自然语言
- ✅ **自动数据获取**: 基于 wechat-cli 自动提取群聊数据
- ✅ **多维度分析**: 活跃度、知识分享、互动深度、内容质量等
- ✅ **Markdown 输出**: 生成格式化的 Markdown 报告

## 前置要求

### 1. 安装 wechat-cli

```bash
npm install -g @canghe_ai/wechat-cli
```

### 2. 初始化 wechat-cli

```bash
# 确保微信正在运行
wechat-cli init
```

### 3. 验证安装

```bash
wechat-cli --version
wechat-cli sessions --limit 5
```

### 支持的时间范围

- **今天**: 当天的数据
- **昨天**: 昨天的数据
- **本周**: 本周一至今
- **上周**: 上周一到上周日
- **本月**: 本月1日至今
- **上月**: 上月1日到上月末
- **自定义**: 2026-04-07、最近3天、最近7天等

### 支持的报告类型

| 报告类型 | 命令关键词 | 说明 |
|----------|-----------|------|
| 群日报 | 群日报、日报、daily-report | 日度运营监控 |
| 成员贡献榜 | 贡献榜、成员榜、member-ranking | 成员激励体系 |
| 内容质量报告 | 质量报告、内容报告、content-quality | 内容运营优化 |
| 成员成长报告 | 成长报告、成员成长、member-growth | 成员运营分析 |
| 综合月报 | 月报、月度报告、monthly-report | 决策层汇报 |

## 报告内容说明

### 1. 群日报 (daily-report)

**用途**: 每日群运营监控

**包含内容**:
- 📊 今日数据（成员数、发言人数、消息总数、活跃度）
- 📨 消息类型分布
- ⏰ 24小时活跃分布图
- 🔥 热门话题 Top 3
- 💬 活跃成员 Top 5
- 📮 今日分享链接
- 📝 金句摘录

### 2. 成员贡献榜 (member-ranking)

**用途**: 成员激励与价值识别

**评分维度**:
- 🎯 发言活跃度 (40%)
- 🔗 知识分享 (25%)
- 💬 互动贡献 (20%)
- ⏰ 守时贡献 (15%)

**包含内容**:
- 🏆 成员贡献总榜 Top 20
- 🎯 发言活跃榜 Top 10
- 🔗 知识分享榜 Top 10
- 💬 互动达人榜 Top 10
- 🌟 守时先锋榜 Top 10
- 🏅 特色称号

### 3. 内容质量报告 (content-quality)

**用途**: 内容运营优化

**质量维度**:
- 内容丰富度
- 知识密度
- 互动深度
- 原创比例

**包含内容**:
- 📊 内容质量概览（5维度评分）
- 📈 内容分布分析
- 🔗 高质量内容榜单
- 💬 优质话题识别
- 🎯 内容质量标签分析
- 📅 优质内容时间线
- 🏆 内容之星
- 📋 质量改进建议

### 4. 成员成长报告 (member-growth)

**用途**: 成员运营与增长分析

**包含内容**:
- 📊 成员成长概览
- 🌟 本周明星成员
  - ⬆️ 飙升榜（潜水→活跃）
  - 🔄 活跃稳定榜
  - ⬇️ 活跃下降榜
- 🎯 成员分层分析
  - 核心活跃层（发言≥20）
  - 潜力成长层（5≤发言<20）
  - 待激活层（发言<5）
- 📈 成长轨迹图
- 🏷️ 成员标签画像
- 🎖️ 本周里程碑
- 📋 成员运营建议

### 5. 综合月报 (monthly-report)

**用途**: 决策层汇报材料

**包含内容**:
- 📊 核心指标概览
- 🔥 本月亮点
  - 热门话题 Top 5
  - 明星成员 Top 10
  - 爆款内容 Top 5
- 📈 趋势洞察
- ⚠️ 问题与建议
- 📅 下月行动计划

## 技术实现

本 Skill 基于 wechat-cli 的以下命令：

| 命令 | 用途 |
|------|------|
| `wechat-cli sessions` | 获取群列表 |
| `wechat-cli history` | 获取聊天记录 |
| `wechat-cli stats` | 获取统计数据 |
| `wechat-cli members` | 获取群成员 |
| `wechat-cli search` | 搜索消息 |

## 注意事项

1. **微信必须运行**: wechat-cli 需要读取微信进程内存获取密钥
2. **时间范围限制**: 单次查询最多返回 1000 条消息
3. **群名称匹配**: 支持完整群名或关键词匹配
4. **数据时效性**: 报告基于 wechat-cli 获取的实时数据

## 故障排查

### 问题 1:wechat-cli 未安装

**错误提示**: `wechat-cli: command not found`

**解决方案**:
```bash
npm install -g @canghe_ai/wechat-cli
```

### 问题 2:未初始化

**错误提示**: `config.json not found`

**解决方案**:
```bash
wechat-cli init
```

### 问题 3:微信未运行

**错误提示**: `无法获取微信进程`

**解决方案**: 
- 打开微信并登录
- 确保 wechat-cli init 时选择的账号当前已登录

### 问题 4:群名不存在

**错误提示**: `无法找到指定的群`

**解决方案**:
- 运行 `wechat-cli sessions --limit 50` 查看可用群
- 使用群名称中的关键词进行匹配
