# PPT 风格提示词生成器 - 使用指南

## 简介

这是一个 PPT 幻灯片风格提示词库，包含 11 种精心设计的视觉风格，可直接用于 nanobanana2 (302.ai) 生成高质量 PPT 图片。

---

## 安装位置

`d:/ProjectAI/CC/CC_record/skills/ppt-style-prompts/`

---

## 快速使用（三步法）

### Step 1: 选择风格并复制 Prompt

打开 [PROMPTS.md](./PROMPTS.md)，选择你喜欢的风格，复制对应的 Prompt。

### Step 2: 修改内容

直接在 Prompt 中修改以下内容：
- `bold title text` → 你的标题
- `subtitle below` → 你的副标题
- `key statistics displayed as cards` → 你的数据/要点

### Step 3: 粘贴生成

将修改后的 Prompt 粘贴到 nanobanana2 生成！

---

## 推荐设置

| 设置项 | 值 |
|--------|-----|
| Model | `gemini-3.1-flash-image-preview` (nanobanana2) |
| Resolution | `2048*1152` (2K 16:9) |
| Aspect Ratio | `16:9` |

## 11 种风格速查

| 风格 | 适用场景 | 特点 |
|------|---------|------|
| 1. 复古波普 | 创意展示、品牌宣传 | 70年代杂志风，粗黑边框，温暖色系 |
| 2. 极简主义 | 企业汇报、产品介绍 | 白底，大量留白，优雅线条 |
| 3. 赛博朋克 | 科技主题、未来感 | 暗色背景，霓虹发光，电路装饰 |
| 4. 新粗野主义 | 个性表达、艺术展示 | 原始设计，粗边框，强烈对比 |
| 5. 酸性设计 Y2K | 潮流内容、年轻受众 | 金属质感，液态形状，全息效果 |
| 6. 现代极简波普 | 社交媒体、轻量内容 | Instagram 风格，柔和粉彩 |
| 7. 瑞士国际主义 | 专业设计、高端展示 | Helvetica 字体，高饱和色块，非对称布局 |
| 8. 暗黑编辑出版 | 深度内容、评论分析 | NYT 周刊风格，白色点阵，衬线字体 |
| 9. 设计蓝图 | 产品文档、技术说明 | Figma 文档风格，青色网格，标注线 |
| 10. 粗野主义 UI | 界面展示、SaaS 产品 | 仪表盘设计，柔和面板，卡片布局 |
| 11. Y2K 像素复古 | 怀旧主题、创意内容 | 90年代像素风，CRT 显示器，等距图标 |

## 内容创作建议

### 标题
- 保持简洁，8 个单词以内
- 例："What is Y Combinator"

### 副标题
- 一行解释，12 个单词以内
- 例："The World's Most Famous Startup Accelerator"

### 关键数据
- 3-5 个数据点
- 例："2005, 4000+ companies, $600B"

### 视觉平衡
- 留白 30%
- 字体层级：标题 > 副标题 > 数据 > 装饰


## 技巧

1. **直接修改文字** - Prompt 中已预留文字位置描述，直接替换即可
2. **保持 Prompt 简洁** - AI 会自动处理文字渲染
3. **始终指定 16:9** - 确保 PPT 格式正确
4. **使用颜色名称 + HEX** - 如 "salmon pink #FF6B6B" 更精确
5. **多试几次** - 同样 Prompt 可能产生不同结果
