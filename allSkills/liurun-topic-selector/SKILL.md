# 刘润公众号选题智能体

## 概述

这是一个专为中国商业内容创作者设计的选题智能体，以"类刘润"风格分析商业事件，提供深度的选题建议和写作角度。

**核心功能**：
1. **新闻扫描**：自动抓取36氪、虎嗅、晚点、界面等中国主流商业媒体的最新报道
2. **智能筛选**：聚焦互联网、新能源、AI、大消费四大领域，关注Tesla、Nvidia等焦点公司
3. **深度分析**：提取关键数据、识别多方观点、发现争议点
4. **选题推荐**：生成"类刘润"风格的Top3选题建议和写作角度

**关注领域**：
- 互联网/TMT
- 汽车/新能源
- AI/大模型
- 大消费/零售

**关注公司**：
- 国际：Tesla、Nvidia、Apple、Microsoft、OpenAI
- 中国：比亚迪、理想、蔚来、小鹏、华为、小米、字节、阿里、腾讯

## 使用方法

### 快速开始

**触发词**：
- "今天有什么商业热点"
- "推荐公众号选题"
- "刘润风格选题"
- "商业新闻分析"
- "抓取商业新闻"

**执行流程**：

Claude会自动执行以下步骤：

#### Step 1: 抓取新闻 (fetcher.py)
```bash
cd CC_record/skills/liurun-topic-selector/scripts
python main.py [hours]
```

默认获取最近48小时内的新闻。

#### Step 2: 分析事件 (analyzer.py)
脚本会自动：
- 提取关键数据和事实
- 识别事件类型（投融资、财报、产品发布等）
- 计算影响力评分
- 提取相关实体（公司名等）

#### Step 3: 选择选题 (selector.py)
基于"类刘润"风格筛选：
- **微观切入**：从小事件看大逻辑
- **深度洞察**：能提炼规律/方法论
- **启发思考**：给读者新认知
- **素材丰富**：有足够事实和数据

#### Step 4: 生成报告 (reporter.py)
生成两份报告：
- `output/daily_report.md` - 完整每日报告
- `output/topics_recommendations.md` - 选题推荐


```json
{
  "sectors": ["互联网", "汽车", "新能源", "AI", "大消费"],
```

## 设计哲学

### 脚本负责机械，Claude负责智能

**脚本处理**：
- 抓取RSS feed
- 过滤时效性
- 提取基础信息
- 计算评分

**Claude处理**：
- 深度事件分析
- 多方观点识别
- 争议点发现
- 写作角度设计

### 类刘润风格特征

1. **微观切入**: 从具体事件/场景入手，不是宏大叙事
2. **拆解逻辑**: 本质是什么，不是什么
3. **生活类比**: 用读者熟悉的事物类比复杂商业概念
4. **启发思考**: 不只是传递信息，更给读者新认知

### 常用标题模式

- "从[事件]看懂[逻辑]"
- "拆解[现象]背后[因素]"
- "[决策]带来何种启发"
- "[公司]的[战略/选择]"

## 常见问题

### Q: 没有抓取到文章？
**A**: 检查以下几点：
1. 网络连接是否正常
2. RSS源是否可用（部分网站可能有反爬措施）
3. 时间范围是否过小（尝试增加到72小时）

### Q: 如何添加新的新闻源？
**A**:
1. 编辑 `config/sources.json`
2. 添加新的源配置
3. 如果是RSS格式，直接使用；如果是API，需要在`parsers/`中添加对应解析器

### Q: 如何调整选题推荐的风格？
**A**:
1. 编辑 `scripts/selector.py`
2. 修改 `_generate_topic_name()` 和 `_generate_writing_angles()` 函数

### Q: 报告中的深度分析需要手动完成吗？
**A**: 是的。脚本负责提取基础信息，深度分析（多方观点、争议点等）需要Claude基于LLM能力完成。这是设计哲学：脚本做机械的事，AI做智能的事。
