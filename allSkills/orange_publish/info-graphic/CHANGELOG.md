## [1.0.0] - 2026-02-28

### Added
- 8 visual styles: Coordinate Blueprint, Retro Pop Grid, Folder Style, Receipt Ticket, Hand-drawn Journal, Detective Archive, Color Block, and Ticket Style
- Support for multiple aspect ratios (16:9, 9:16, 1:1, 4:3, 3:4)
- 4K ultra high definition output
- Automatic language detection and preservation
- Style template system with editable markdown files
