# Life Reset - 一日人生重启方案

> 基于 Dan Koe 的深度心理学框架，帮助你彻底审视并重构自己的生活路线。

## 简介

这是一个科学的生活指导和心灵助手 skill，通过结构化的反思框架帮助你：

- 🔍 **揭示隐藏动机** - 发现真正驱动你行为的无意识目标
- 💔 **打破身份循环** - 识别并打破限制你的条件反射模式
- 🎯 **重构生活方向** - 创建清晰的愿景和可执行的行动框架
- 🎮 **游戏化人生** - 把生活变成电子游戏，享受进步的过程

## 使用场景

| 场景 | 说明 |
|------|------|
| 📅 年度反思 | 新年、生日、季度回顾时使用 |
| 🔄 生活迷茫期 | 感到停滞、缺乏方向时 |
| ⚠️ 重大转折前 | 辞职、创业、迁居、转行前 |
| 💔 反复失败后 | 陷入同样的失败模式时 |
| 🧘 自我审视 | 定期深度自我对话 |

## 工作模式

### 1. 完整一日反思模式

适合预留完整一天的深度反思。

**流程：**
```
🌅 早上 (45分钟) → 心理挖掘：揭示痛苦、创建反愿景、创建愿景
☀️ 全天 (持续)   → 打断自动驾驶：定时提醒问题
🌙 晚上 (45分钟) → 综合洞察：整合发现、创建行动框架
```

### 2. 问题卡片模式

适合碎片化时间的针对性思考。

**使用方式：**
- 随机抽取一个问题进行深度思考
- 每天固定时间回答1-2个问题
- 用于日记、晨间反思或睡前思考

### 3. 在线辅导模式

与 AI 助手对话式完成整个流程。

**特点：**
- 引导式提问，逐步深入
- 根据你的回答调整问题
- 帮助你识别盲点和矛盾
- 生成个性化的行动建议

## 核心理论

### 身份循环

```
设定目标 → 感知现实 → 选择性注意 → 行动
→ 重复至自动化 → 成为身份 → 维护身份 → 重启循环
```

**突破点：** 有意识地打断"维护身份"这一步

### 意识发展的9个阶段

| 阶段 | 名称 | 特征 |
|------|------|------|
| 1 | 冲动型 | 冲动与行动无分离 |
| 2 | 自我保护型 | 学会保护自己、隐藏弱点 |
| 3 | 从众型 | 你就是你的群体 |
| 4 | 自我觉察型 | 注意到内心与外在不符 |
| 5 | 尽责型 | 建立自己的原则体系 |
| 6 | 个人主义者型 | 注意到原则由情境塑造 |
| 7 | 战略家型 | 与系统合作同时保持觉察 |
| 8 | 建构觉察型 | 将所有框架视为有用的虚构 |
| 9 | 合一型 | 自我与生活分离消融 |

大多数人在 4-8 之间徘徊。

## 六要素生活游戏

完成反思后，将结果组织成六个部分：

| 要素 | 问题 | 游戏类比 |
|------|------|----------|
| 反愿景 | 我永远不想过什么样的生活？ | 赌注 |
| 愿景 | 我理想中的生活是什么样的？ | 胜利条件 |
| 一年目标 | 一年后我要达成什么？ | 主任务 |
| 一个月项目 | 这个月我要学什么/做什么？ | Boss战 |
| 每日杠杆 | 每天最重要的2-3件事是什么？ | 日常任务 |
| 约束 | 我绝不妥协的底线是什么？ | 规则 |

## 使用建议

1. **诚实是前提** - 欺骗自己只会浪费时间
2. **不要外包思考** - AI 可以引导，但无法替你感受和选择
3. **痛苦是好事** - 对现状的不适是改变的动力
4. **愿景会演变** - 现在的答案不必是完美的
5. **行动最重要** - 没有行动，洞察一文不值
