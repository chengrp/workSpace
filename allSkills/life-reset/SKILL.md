# Life Reset - 一日内人生重启方案

基于 Dan Koe 的《如何在一天内改变你的人生》设计的一日深度反思框架。

## 核心理念

**真正的改变不是改变行为，而是改变"你是谁"。**

大多数人失败的原因：
- ❌ 在"腐烂的地基"上建造美好（只改变行为，不改变身份）
- ❌ 话语说一套，行动做另一套（无意识目标在主导）
- ❌ 被条件反射困住，维护旧身份以保持心理一致性

## 成功的三个阶段

```
失调 → 不确定性 → 发现
```

1. **失调** - 对当前生活感到厌倦，觉得"不属于这里"
2. **不确定性** - 不知道接下来会发生什么，实验或迷失
3. **发现** - 找到真正想要追求的东西，6个月内取得6年的进展

---

## 使用时机

- 年度反思/新年规划
- 感到生活停滞、缺乏方向
- 重大人生转折前（辞职、创业、迁居）
- 陷入反复失败的模式
- 想要深度审视自己的人生路线

## 完整方案：一日反思流程

### 🌅 早上（30-45分钟）- 心理挖掘

**目标：** 创建新的感知框架，揭示隐藏动机

#### 步骤1：揭示当前痛苦（15分钟）

> 不要外包给AI，自己用笔和纸回答

1. **你已经学会忍受的那种沉闷而持续的抱怨是什么？**
   - 不是深层痛苦，而是你已经学会容忍的东西
   - （如果你不讨厌它，你就会容忍它）

2. **你反复抱怨但从未真正改变的是什么？**
   - 写出过去一年你表达最多的三个抱怨

3. **对于每个抱怨：观察你行为（而不是你话语）的人会得出什么结论？**
   - 说你实际上想要什么？

4. **关于你当前的生活，向你深表尊重的人承认什么真相会让人无法忍受？**

#### 步骤2：创建反愿景（15分钟）

> 利用负面能量，将努力导向积极方向

1. **如果接下来五年绝对没有任何改变，描述一个普通的星期二。**
   - 你在哪里醒来？身体感觉怎样？
   - 想到的第一件事是什么？谁在你身边？
   - 早上9点到下午6点之间你在做什么？
   - 晚上10点你感觉如何？

2. **现在再做一次，但针对十年。**
   - 你错过了什么？什么机会关闭了？
   - 谁放弃了你？
   - 当你不在房间时，人们怎么评价你？

3. **你在生命的尽头。你过着安全的版本。你从未打破这个模式。代价是什么？**
   - 你从未让自己感受、尝试或成为什么？

4. **谁已经在过你刚才描述的未来？**
   - 同一条道路上领先你五年、十年、二十年的人
   - 当想到要成为他们时，你有什么感觉？

5. **要真正改变，你必须放弃什么身份？**
   - （"我是那种会……的人"）
   - 不再做那个人在社交上会付出什么代价？

6. **你没有改变的最令人尴尬的原因是什么？**
   - 那个让你听起来软弱、害怕或懒惰，而不是合理的原因？

7. **如果你当前的行为是一种自我保护，你到底在保护什么？**
   - 那种保护让你付出了什么代价？

#### 步骤3：创建愿景（15分钟）

> 忘记可行性，创造你真正想要的生活

1. **如果你弹指之间就能在三年内过上不同的生活（不是现实的，而是你真正想要的），一个普通的星期二是什么样的？**
   - 与反愿景问题相同程度的细节

2. **你必须相信自己什么，那个生活才会感觉自然而不是强迫？**
   - 写出身份声明："我是那种会……的人"

3. **如果你已经是那个人，你这周会做哪一件事？**

---

### ☀️ 全天 - 打断自动驾驶

**目标：** 破坏无意识模式，强制有意识思考

#### 定时提醒问题

设置6个随机时间提醒，包含这些问题：

| 时间 | 问题 |
|------|------|
| 上午11:00 | 我现在通过正在做的事在逃避什么？ |
| 下午1:30 | 如果有人拍下过去两个小时，他们会得出什么结论，说我想从生活中得到什么？ |
| 下午3:15 | 我是在向我讨厌的生活还是我想要的生活推进？ |
| 下午5:00 | 什么是我假装不重要最重要的事？ |
| 晚上7:30 | 我今天做什么是出于身份保护而不是真正的欲望？ |
| 晚上9:00 | 今天什么时候我感觉最活着？什么时候我感觉最死？ |

#### 通勤/散步时反思

在通勤、散步或躺着的时候思考：

- 如果我不再需要人们把我看作【你在反愿景问题5中写的身份】，会发生什么？
- 在我的生活中，我在哪里用活力换取了安全？
- 我想成为的那个人的最小版本的什么，我明天就可以成为？

---

### 🌙 晚上（30-45分钟）- 综合洞察

**目标：** 整合洞察，创建行动框架

#### 综合反思

1. **今天之后，关于你为什么一直被困住，什么感觉最真实？**

2. **真正的敌人是什么？**
   - 清楚地命名它
   - 不是环境，不是其他人
   - 一直在主导的内部模式或信念

3. **写一句话概括你拒绝让你的生活变成什么。**
   - 这是你的**压缩反愿景**
   - 读它时应该让你有感觉

4. **写一句话概括你正在建设什么，知道它会演变。**
   - 这是你的**愿景最小可行产品**

#### 创建时间透镜目标

> 目标不是终点线，而是观点/透镜，帮助你进入正确的心理状态

1. **一年透镜：一年内什么必须为真，你才能知道你打破了旧的模式？**
   - 一件具体的事

2. **一个月透镜：一个月内什么必须为真，才能让一年透镜保持可能？**

3. **每日透镜：明天你可以时间阻塞的2-3个行动是什么？**
   - 正在成为的那个人会简单地做那些事

## 最终框架：把生活变成电子游戏

将所有洞察组织成6个组成部分：

| 组成部分 | 说明 | 游戏类比 |
|----------|------|----------|
| **反愿景** | 什么是我的存在之痛，或我永远不想再经历的生活？ | 赌注：输了的后果 |
| **愿景** | 什么是理想的我想要的生活，随着我向它努力可以不断完善？ | 胜利条件：怎么赢 |
| **一年目标** | 一年后我的生活会是什么样的？ | 主任务：唯一优先事项 |
| **一个月项目** | 我需要学习什么？获得什么技能？构建什么？ | Boss战：获得经验值 |
| **每日杠杆** | 什么是优先级、能推动进程的任务？ | 日常任务：解锁机会 |
| **约束** | 实现愿景时，我有什么不愿意牺牲的？ | 规则：激发创造力 |

## 理论基础

### 身份的解剖学

```
设定目标 → 感知现实 → 只注意到"重要"信息 → 朝目标行动
→ 重复直到自动化 → 成为身份的一部分 → 维护身份保持一致性
→ 新目标重启循环
```

**打破循环：** 必须有意识地意识到第6步和第7步之间发生了什么

### 意识发展的9个阶段

| 阶段 | 特征 | 典型表现 |
|------|------|----------|
| 1. 冲动型 | 冲动与行动无分离，非黑即白 | 蹒跚学步的孩子生气打人 |
| 2. 自我保护型 | 学会保护自己，隐藏弱点 | 孩子藏成绩单、在家务撒谎 |
| 3. 从众型 | 你就是你的群体 | 无法理解为什么不和家人投一样的票 |
| 4. 自我觉察型 | 注意到内心与外在不符 | 坐在教堂里怀疑自己是否相信 |
| 5. 尽责型 | 建立自己的原则体系 | 研究后离开家庭宗教，制定职业计划 |
| 6. 个人主义者型 | 注意到原则由情境塑造 | 意识到政治观点与成长地有关 |
| 7. 战略家型 | 在意识到自己参与其中的同时与系统合作 | 领导组织时主动质疑自己的盲点 |
| 8. 建构觉察型 | 所有框架（包括身份）都是有用的虚构 | 温和幽默地看待自己扮演的角色 |
| 9. 合一型 | 自我与生活的分离消融 | 工作、休息、玩耍感觉是同一回事 |

大多数人处于4-8之间。**通过任何阶段都遵循一个模式。**

### 控制论：智力的定义

**智力 = 从生活中得到你想要的东西**

控制论循环：
```
有目标 → 朝目标行动 → 感知当前位置 → 与目标比较 → 根据反馈再次行动
```

高智力的标志：
- 迭代、坚持和理解大局的能力
- 意识到任何问题在足够大的时间尺度上都可以解决
- 理解想法有层次，不能一步从纸莎草纸跳到谷歌文档

## 重要提示

1. **不要外包思考** - 这些问题必须自己回答，AI无法替你完成心理挖掘

2. **诚实是痛苦的** - 如实回答会让你对当前生活方式感到不适，甚至厌恶——这是好事

3. **愿景会演变** - 你的愿景是最小可行产品，会随着时间和经历变得更清晰

4. **目标不是终点** - 目标是透镜/观点，帮助你进入正确的心理状态

5. **乐趣在于进步** - 没有终点线，进步本身就是回报

## 使用此技能的最佳方式

1. **预留完整的一天** - 不要被打断，关闭通知
2. **准备工具** - 纸笔或笔记本（不要只用电子设备）
3. **诚实面对自己** - 这是一切有效的前提
4. **按顺序执行** - 早上→全天→晚上
5. **保存结果** - 将你的6个游戏要素写在显眼的地方
