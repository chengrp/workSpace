```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                              │

└──────────────────┬──────────────────────────────────────────────────────┘
                   │
                   ▼
        ┌─────────────────────────────────┐
        │  阶段 1: 需求澄清      │
        │ 挖需求：内容/场景/受众/字多字少 │
        │                              │
        └──────────────────┬─────────────┘
                         │
                         ▼
              ┌──────────────────────────┐
              │  阶段 2: 配图规划  │
              │ 拆内容→定图清单      │
              │                         │
              └──────────────────┬────────┘
                               │
                                ▼
                     ┌──────────────────────────┐
                     │ 阶段 3: 文案定稿  │
                     │ 逐字定稿"图上写什么" │
                     │                       │
                     └──────────────────┬─────────┘
                                    │
                                    ▼
                        ┌─────────────────────────────┐
                        │ 阶段 3.5: 风格选择 │

                        │ - craft-handmade        │

                        │                        │
                        └────────────┬───────────┘
                                   │
                        🛑 AskUserQuestion 工具等待用户选择
                                   │
                                   ▼
                        ┌─────────────────────────────┐
                        │ 阶段 4: 提示词封装  │
                        │ 封装成可复制提示词      │
                        │                         │
                        └──────────────────┬─────────┘
                                       │
                                       ▼
                            ┌──────────────────────────┐
                            │ 阶段 5: 迭代润色  │
```

## 关键决策点

| 阶段 | 用户需要确认的内容 |
|------|-----------------|
| 阶段 1 | 内容是什么？用在什么场景？谁来看？要字多还是字少？ |
| 阶段 2 | 需要几张图？每张图讲什么？ |
| 阶段 3 | 图上具体写哪些字？（逐字确认） |
| **🛑 阶段 3.5** | **选择 17 种风格之一**（强制停止） |
| 阶段 4 | 确认提示词，选择生图 API |
| 阶段 5 | 字太多？风格不对？需要迭代 |
